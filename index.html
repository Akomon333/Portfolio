Spectral Analysis of Meteorite Composition via Computer Vision

Flask API OpenCV XGBoost

Problem: Production API analyzing meteorite imagery to estimate mineral composition (Ni, Ir, Au), operational risk, and surface topology. Built under strict time constraints (168 hours) as a feasibility study for automated asteroid mining assessment.

  • Segmentation: Otsu's method for background removal, RGB mean extraction
  • Inference: XGBoost regressor for precious metal concentrations with graceful degradation to deterministic heuristics if model unavailable
  • Topology: 128×128 displacement map via SimpleBlobDetector + Gaussian smoothing for 3D terrain generation
Output Schema
composition: {
  nickel_pct: 10.5,
  iridium_ppm: 3.2,
  gold_ppm: 0.8
},
economics: {
  value_usd_per_ton: 2500.0,
  mission_risk: 0.45
}
Time: O(n log n) | Space: O(n²)
